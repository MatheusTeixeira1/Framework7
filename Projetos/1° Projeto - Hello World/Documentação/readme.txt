Oque é esse projeto?
   - Este projeto é o clássico "Hello World", desenvolvido para quebrar a maldição e garantir um 
     começo seguro (eu não arriscaria, e você também não deveria!). De maneira simples e objetiva, 
     este projeto tem como objetivo ajudar a se familiarizar com o padrão de design do Framework7, 
     entender o processo de criação de um projeto do zero e explorar alguns dos componentes oferecidos
     pelo framework.

Como começar?
   - Requisitos:
	- Node.js

   - Intalação do Framework7 CLI (O Framework7 CLI é uma ferramenta que facilita a criação de novos projetos.)
	- npm install -g framework7-cli (Para instalação)
	- framework7 --version (Para conferir)

   - Na pasta que vai localizar o projeto:
	- framework7 create (Para inicialização facilitada do projeto escolha as seguintes opções de instalação)
	   - > Simple web app
	   - > (Digite o nome do projeto)
	   - > Framework7 Core
	   - > Single View
	   - > Vite (recommended)
	   - > No, i am good with CSS
	   - > No, use default color theme
	   - > Yes, include icon fonts
	- npm install (Para instalar as dependências)
	- npm start (Para iniciar o servidor)
	- depois disso o prompt de comando deve ficar assim:
	     VITE v6.0.5  ready in 2614 ms

 	     ➜  Local:   http://localhost:5173/ 	     
 	     ➜  Network: http://192.168.1.77:5173/
 	     ➜  press h + enter to show help

   - Agora é só explorar o projeto. ;)
