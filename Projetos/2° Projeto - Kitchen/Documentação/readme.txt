Oque é esse projeto?
   - Este projeto é um modelo disponibilizado pelos desenvolvedores do Framework7 
     ser usado como documentação de referência para o desenvolvimento de aplicações
     usando o framework.

Como instalar?
   - Requisitos:
	- Git e Github
	- Git Bash
	- Node.js
	- npm do None.js

   - Na pasta que se localiza o projeto abra o Git Bash e digite:
	- git clone https://github.com/framework7io/framework7.git (Para clonar o repositorio)
	- npm install (Para instalar as dependencias necessarias para o projeto)
	- npm run core (Para rodar o projeto)
	- para fechar o servidor pressione Crtl + c.
